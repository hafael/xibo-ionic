# Servidor CMS

O servidor do CMS (Content Management System) está configurado para ser acessado localmente no seguinte endereço:

- **Endereço:** `localhost`
- **Porta:** `80`

Este setup é utilizado para desenvolvimento e testes, permitindo a integração com o projeto Xibo-Ionic em um ambiente local.
